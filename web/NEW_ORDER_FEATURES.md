### Best Practices

1. **Keep Dashboard Open**
   - Leave dashboard open in browser tab
   - Even if not actively viewing, notifications will alert you

2. **Enable Sound**
   - Ensure device volume is on
   - Click page after load (browser security requirement)

3. **Watch for NEW Badge**
   - Bright green badge is hard to miss
   - Disappears after 10 seconds (act fast!)

4. **Use Pending Tab**
   - Default tab shows only orders needing action
   - Switch to other tabs to check status

5. **Search Orders**
   - Use search bar for customer name/phone
   - Filter by status for specific views

### Troubleshooting

**No sound?**
- Check device volume
- Click anywhere on page first (browser requirement)
- Check browser permissions

**Not seeing new orders?**
- Check internet connection
- Click "Refresh" button
- Check if logged in correctly

**NEW badge stuck?**
- Will auto-remove after 10s
- Doesn't affect functionality
- Refresh page if needed
