# KnockBites Email Templates for Supabase

## Setup Instructions

1. Go to **Supabase Dashboard** → **Authentication** → **Email Templates**
2. Click on each template type and paste the corresponding HTML
3. Update the Subject line as noted below
4. Save changes

## Templates & Subjects

| Template | Subject Line |
|----------|--------------|
| Confirm Sign Up | `Welcome to KnockBites! Please confirm your email` |
| Reset Password | `Reset your KnockBites password` |
| Magic Link | `Your KnockBites sign-in link` |
| Change Email | `Confirm your new email address - KnockBites` |
| Invite User | `You're invited to join KnockBites!` |

## Brand Colors

- **Primary Orange:** `#F5A623`
- **Secondary Orange:** `#FF9500`
